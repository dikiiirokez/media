The hobby project

It represents a window-based GUI + filesystem on frontend.
The site you see is built on this engine.
Sources on GitHub The idea is:
a) implement a full-fledged windowed user interface on the front end, replicating features as in Windows 10;
b) provide the system with a data server equipped with an abstract filesystem and an interface for data operations;
c) organize this in a form of finite state machine.
Made it for fun - just was interested in such engineering challenge as "implement Windows on React".
Ecosystem: React, RTK, TS, SCSS, ViteJS.
