The hobby project

It is window-based GUI + some filesystem on frontend.
The site you see is built on this engine.
Made this for fun - just was interested in such engineering challenge as "implement Windows on React".
The idea is:
- to implement on front-end a full-fledged window user interfece, where is possible to do everything like in Windows 10;
- provide this with a data server equipped with abstract filesystem and interface for operating it;
- organize this all in a form of finite state machine
Ecosystem: React, RTK, TS, SCSS, ViteJS.
