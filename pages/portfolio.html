The hobby project

It represents a window-based GUI + filesystem on frontend.
The site you see is built on this engine.
Sources on GitHub
The idea is:
a) implement a windowed user interface on the front-end, and equip it with features similar to Windows 10;
b) provide the application with a data server equipped with an abstract filesystem, functionality to operate the data tree, and API for the actions on data;
c) rganize the engine in a form of finite state machine.
Made it for fun - just was interested in such engineering challenge as "implement Windows on React".
Ecosystem: React, RTK, TS, SCSS, ViteJS.
